## Composants

- [PowerShell 7](https://github.com/PowerShell/PowerShell) - Shell moderne cross-platform
- [Terminal-Icons](https://github.com/devblackops/Terminal-Icons) - Icones colorees dans le terminal
- [PSReadLine](https://github.com/PowerShell/PSReadLine) - Autocompletion intelligente
- [Zoxide](https://github.com/ajeetdsouza/zoxide) - Navigation rapide entre dossiers
- [Yazi](https://github.com/sxyazi/yazi) - Gestionnaire de fichiers TUI
- [Vim](https://github.com/vim/vim) - Editeur de texte modal
- [CascadiaCode Nerd Font](https://github.com/ryanoasis/nerd-fonts) - Police avec icones integrees

## Prerequis

- Windows 10/11
- [winget](https://apps.microsoft.com/detail/9NBLGGH4NNS1) (App Installer)
- Droits administrateur (optionnel, pour installer la police)
